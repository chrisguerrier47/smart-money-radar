# Smart Money Radar

Aggregates insider trades, activist filings, and news into conviction scores using SEC EDGAR, Finnhub, and Gemini AI.

## APIs Used
| Service | Purpose | Cost |
|---------|---------|------|
| SEC EDGAR | Insider buys + activist 13D/G | Free (email only) |
| Finnhub | Insider transactions + news | Free tier |
| Google Gemini | LLM signal analysis | Pay per use |

## Local Setup

```bash
cp .env.example .env
# Fill in your 3 API keys, then:
docker-compose up --build
```

- Frontend: http://localhost:5173
- Backend API: http://localhost:8000
- API docs: http://localhost:8000/docs

## Railway Deployment

1. Push to GitHub (see bash instructions below)
2. railway.app → New Project → Deploy from GitHub
3. Select your repo, set root to `backend/`
4. Add PostgreSQL plugin
5. Add environment variables (see Railway Variables below)
6. Deploy frontend as second Railway service, root `frontend/`

## Railway Environment Variables (backend)

```
DATABASE_URL        = auto-generated by Railway PostgreSQL plugin
SEC_USER_AGENT      = your@email.com
FINNHUB_API_KEY     = your_finnhub_key
GEMINI_API_KEY      = your_gemini_key
ENV                 = production
```
